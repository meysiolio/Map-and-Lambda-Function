**Input Format**

One line of input: an integer ***N***.  

**Output Format**

A list on a single line containing the cubes of the first ***N*** fibonacci numbers.  

**Sample Input**
```
5
```
**Sample Output**
```
[0, 1, 1, 8, 27]
```
**Explanation**

The first **5** fibonacci numbers are **[0,1,1,2,3]**, and their cubes are **[0,1,1,9,27]**.
